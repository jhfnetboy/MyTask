# MyTask
It is another version of OpenTasks protocol prototype version.

## Features
1. It is a AI Application
2. It is a x402 application
3. It is a permissless application
4. It is four players flow: Task Publisher(Sponsor), Taskor(who do the task), Target item supplier(who provide the thing Taskor want) and Jury(who help to audit the quariles)
5. It is OpenPNTs adaptable application(accept all ERC-20 obey the OpenPNTs protocol)
6. It is Open Source with MIT license.
